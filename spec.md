# やることポケット 仕様書
## 1. アプリ名
やることポケット
## 2. 目的
- 対象：個人用
- 解決したいこと：タスクを必ず実行するため
## 3. 画面一覧
### ① ホーム画面（リスト）
- 画面上部に説明文：「ボールが来たらすぐに登録する」
- 右上の「＋」ボタン → タスク追加画面を表示
  - 「タスクの内容を入力してください」に入力
- 入力したタスクは一覧に表示
- 各タスクの右側に4つのアイコンを表示：📌 ☑️ 🗓️ 💡
- チェックボックスを押すと「タスクを完了しました。楽勝🖊️」と表示＋「元に戻す」ボタン
- ボトムナビゲーション（5つ）：
  - リスト ／ 📌軽め ／ ☑️重め ／ 🗓️スケジュール ／ 💡アイデア
### ② 📌 軽めのタスク
- 一覧で📌を押したタスクがここに表示
- 説明文：「いつもこの中は空にすること」
### ③ ☑️ 重めのタスク
- 一覧で☑️を押したタスクがここに表示
- 説明文：「小さく分解させる」＋ マインドマイスターのログインURL
- タスクに期限（日付のみ）を入力・表示できる
### ④ 🗓️ スケジュール
- 一覧で🗓️を押したタスクがここに表示
- GoogleカレンダーのURLを表示し、登録を促す
### ⑤ 💡 アイデア
- 一覧で💡を押したタスクがここに表示
- マインドマイスターのログインURLを表示
- AIの活用を促す
## 4. 使う技術
- HTML / CSS / JavaScript（1ファイル構成）
